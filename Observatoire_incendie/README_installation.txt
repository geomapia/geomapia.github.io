Installation GitHub Pages
=========================

1. Décompressez ce dossier.
2. Envoyez les fichiers suivants à la racine du dépôt GitHub :
   - index.html
   - apropos.html
   - tableau-de-bord.html
   - methodologie.html
   - contact.html
3. Dans GitHub : Settings > Pages > Deploy from a branch > main > /root > Save.
4. Ouvrez : https://geomapia.github.io/Observatoire-incendie/
