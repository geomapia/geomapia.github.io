Présentation des ressources

Cette page regroupe les ressources de test nécessaires pour prendre en main les applications Geomap IA. Les fichiers proposés permettent aux utilisateurs de tester rapidement l’importation de données, les traitements, la visualisation cartographique et les fonctionnalités d’export.
